Music Genre Classification v.0.1

First of all, check the python libraries on your computer and install them if they are missing.

-librosa
-pandas
-numpy
-sklearn
-keras
-kivy

You can use terminal (linux terminal, powershell etc.) for execute.

You should execute "music_genre_classification.py"

1. It welcomes us with an interface where we will choose music file

2. Choose music

3. On this page, there are buttons with the names of the 4 classification methods that we have processed in our project.

4. If select any of them, classification will be done with that method and which type of music it belongs to will be listed with percentage results.
